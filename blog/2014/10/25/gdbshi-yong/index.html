Gdb使用

Oct 25th, 2014

常用命令

  1. r 运行程序
  2. c 当中断后继续运行
  3. n 单步运行（跳过函数）
  4. s 单步运行（进入函数）
  5. q 退出gdb
  6. l 显示代码

    断点

  7. 函数中设置断点：b XXX.c: n 在XXX.c的第n行设置断点
  8. 条件断点： b XXX.c :n if cmd 当cmd为true时在XXX.c的第n行停

    打印

  9. p XX 打印XX的值
  10. 打印数组 p *a@len 打印数组a中的前len个数值
